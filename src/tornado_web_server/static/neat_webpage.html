About the Device

iPlant is a plant monitoring system that aims to collect measurements of key conditions of the environment surrounding a home plant.

The device collects the temperature, humidity, light intensity and soil moisture. Once this data is collected, it is sent to a web server where it is stored, analysed, and displayed to the user in real-time.

iPlant is designed to minimize power consumption, with batteries being able to last a long time.

How to use the Device

iPlant is simple to use, with few steps needed to operate the device.

After powering the product with batteries, the iPlant device is ready the be put in the ground. The user can see the sensor readings on the iPlant website in real time.
